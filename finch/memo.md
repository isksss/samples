# Finchで遊んでみる。
[runfinch/finch](https://github.com/runfinch/finch)が公開されて少し経って色々と情報が出回り始めたので軽く触れて遊んでみる。  
ほぼDockerと変わらない使用感。  


## how to

### バージョン表示
```
% finch version
```

### 仮想マシンの初期化・実行
```
% finch vm init
```

## compose
```
% finch compose up -d
```
